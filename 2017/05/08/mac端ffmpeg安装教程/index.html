mac端ffmpeg安装教程

发表于 2017-05-08

通过homebrew安装，未安装的homebrew的自行google

一、安装ffmpeg

  1. 安装ffmepeg

    brew install ffmpeg

  2. 查看ffmpeg信息

    brew info ffmpeg

  3. 升级ffmpeg

    brew update && brew upgrade ffmpeg

二、使用教程

将某种格式的视频转化成另一种格式的视频

ffmpeg -i test.flv -b:v 640k test.mp4
